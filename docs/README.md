This directory contains a simple static site for the Bridges repository. To publish the site:

1. Go to the repository Settings -> Pages.
2. Under "Source", select the "Main" branch and the `/docs` folder.
3. Save. Your site will be available at `https://<your-username>.github.io/Bridges/` shortly.

You can edit `docs/index.html` and `docs/styles.css` to customize the site.
